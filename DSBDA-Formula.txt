Formula
A. Logistic Regression
	1. TN = cm[0][0]
	2. FP = cm[0][1]
	3. FN = cm[1][0]
	4. TP = cm[1][1]

B. Naive Bayes
	1. TP = cm[0][0]
	2. FP = cm[1][0] + cm[2][0]
	3. FN = cm[0][1] + cm[0][2]
	4. TN = cm[1][1] + cm[1][2] + cm[2][1] + cm[2][2]

- accuracy = (TP + TN) / (TP + TN + FP + FN)
- error rate = (FP + FN) / (TP + TN + FP + FN)
- precision = TP / (TP + FP)
- recall = TP / (TP + FN)
